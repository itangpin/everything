##方案
###根Node与一般Node的区别
创建Node的时候，该Node的name是否显示是可选的，rootNode就是隐藏name的Node
###构造函数里做的事情
不要把所有事情都在Node实例化的时候来做，这样的话，将一个Node作为根节点的时候，就无法操作了
